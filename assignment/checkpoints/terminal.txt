# 8 - Why would you run the command git status?
The git status displays the state of the working directory. it also shows you the files that have not
being tracked.
It lets you know the changes that have been made, and also the changes that have not being
made.
# 9 - What does the command git add . do?
git add . will add everything to git that is in that current folder. Once you use git add . the files will
no longer be untracked, and they will change to be committed.
The commit is what actually saves the version of the folder.
# 10 - What is the command to push your committed changes to your repo on GitHub?
In order to send files via GitHub you have to use git push -u origin master, and it will create a
location on GitHub.
